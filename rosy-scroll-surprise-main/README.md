# Surprise Reveal

A special surprise reveal application built with modern web technologies.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Step 1: Install the necessary dependencies.
npm install

# Step 2: Start the development server.
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

You can deploy this project using platforms like Vercel, Netlify, or GitHub Pages.
- For Vercel/Netlify: Connect your repository and it will automatically detect the Vite environment.
- For Static Hosting: Run `npm run build` and deploy the contents of the `dist` folder.
